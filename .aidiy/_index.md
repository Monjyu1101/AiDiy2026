# .aidiy インデックス

このフォルダは、AiDiy2026 プロジェクト専用の修正知見を蓄積するための場所です。

## 使い方
- ファイル操作を伴う修正前に、この `_index.md` を確認する
- 類似修正があれば、対応メモを読んで知見を再利用する
- 修正完了後は、必要に応じて `_最終変更.md` とテーマ別メモを更新する

## ファイル一覧
- `_最終変更.md`
  - 直近の修正内容の要約
- `新たなCLI追加時の作業手順.md`
  - 新しい code CLI を AiDiy に追加する時の作業手順

## 更新ルール
- 単なる作業ログではなく、次回の修正に再利用できる知見を残す
- 同じテーマの修正は、可能なら既存メモへ統合する
- `.aidiy` はこのプロジェクト専用とし、他プロジェクトの知見と混在させない
