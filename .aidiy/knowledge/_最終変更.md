# 最終変更

更新日: 2026-04-23

## 今回整理した内容

- `frontend_web` の `Xネコ` を `oneko.js` 互換のマスコット実装へ変更した
  - `react-neko` は npm registry に存在しないため、React 依存は追加しない方針に戻した
  - Vue コンポーネント内で 32px スプライトを切り替え、カーソル追跡・待機・警戒・居眠り・壁/自分ひっかき状態を処理する
  - iframe ラッパーではなく、`Xネコ.vue` で直接描画する
- `frontend_web/public/Xネコ/oneko.gif` を追加した
  - 出典は `adryd325/oneko.js`
  - MIT ライセンス資産のため `frontend_web/public/Xネコ/NOTICE.md` に出典とライセンスを記録した
- 表示安定化のため、実際の Vue 画面では `frontend_web/src/assets/oneko.gif` を import して使う
  - 日本語 `public` パス解決と親レイアウト高さ 0 の両方を避けるため、`Xネコ.vue` には `min-height` も設定する
- `.aidiy/knowledge/X系静的画面追加.md` を更新し、X系でもマスコット系は Vue 直実装を許容する方針を追記した
