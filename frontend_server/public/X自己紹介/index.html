Architecture

🗄️ バックエンド

FastAPI + SQLAlchemy + SQLite。デュアルサーバー（Core: 8091 / Apps: 8092）。POST中心API設計、統一レスポンス形式。Alembic不使用のシンプルなスキーマ管理

🎨 フロントエンド

Vue 3 Composition API + Vite + TypeScript + Pinia。qTublerカスタムテーブル、共通ダイアログシステム。UIフレームワーク不使用の軽量設計

🇯🇵 日本語ファースト

テーブル名・カラム名・APIエンドポイント・JSONキー・Vueコンポーネント全て日本語。ビジネスドメインとコードの完全一致

🔐 認証システム

JWT認証（HS256、60分有効期限）。Pinia + Axios interceptorによる自動トークン管理。401エラー時の自動ログアウト

🔢 C採番システム

AUTOINCREMENT不使用のカスタムID生成。C採番テーブルで一元管理。トランザクション保護された予測可能なID

🔄 Reboot機構

reboot1.txt/reboot2.txt による内部再起動システム。ゼロダウンタイムでのコード更新。_start.pyによる自動プロセス監視
