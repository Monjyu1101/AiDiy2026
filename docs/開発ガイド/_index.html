AiDiy2026 開発ガイド

FastAPI + Vue 3 で構築する日本語ファーストの業務アプリケーション開発ガイド

閲覧方法: HTMLファイルを直接ブラウザで開くか、VS Code の Live Server 拡張機能を使用してください。

サーバー起動: python _start.py でフロントエンド (8090) + Core API (8091) + Apps API (8092) が起動します。

ガイド一覧

  • 00. このプロジェクトの歩き方 入門 プロジェクト概要、技術スタック、セットアップ、バックエンド・フロントエンド・DB管理・テスト・運用の入門ガイド (全10章)
  • 01. 環境構築ハンズオン ハンズオン _setup.py / _start.py を使った開発環境のセットアップ手順
  • 02. 設計 ハンズオン DB設計、テーブル命名規則 (C/M/T/V/S/A/X系)、API設計、ディレクトリ構成
  • 03. バックエンド開発 ハンズオン Schema / Model / CRUD / Router の実装手順、Migration、動作確認、ID運用ポリシー
  • 04. フロントエンド開発 ハンズオン Vue 3 + Vite + TypeScript での一覧・編集画面の作成、API連携、ルーター登録

リファレンス

  • 11. コーディングルール リファレンス 命名規則、ファイル構成、コーディング規約、日本語ファーストの実装ルール
  • 12. フロントエンド画面追加例 リファレンス CRUD画面 (一覧・編集) の追加手順、qTublerコンポーネントの使い方
