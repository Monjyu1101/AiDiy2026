AiDiy 2026年版について
  • フロント: 8090 / バックエンド: 8091(Core) / 8092(Apps)
  • DB: SQLiteファイル backend_server/_data/AiDiy/database.db
  • マイグレーションツール(Alembic)は使わず、DB削除で再構築

このシステムは何をする？

AiDiyは「管理画面中心の業務アプリ」をサクッと構築できる仕組みです。
日本語のテーブル名/JSON/URLで、現場の言葉にそのまま合わせて使います。
環境構築 → 画面確認 → API確認 → テーブル追加 → 画面追加 の順に学びます。

3つの基本原則

  2. シンプル構成 — 余計な抽象化をしない、見れば分かるコード
  3. AI活用前提 — AIエージェントが読みやすい構造設計

クイックスタート

最短で動かす手順です。

# プロジェクトに移動
cd workspace/AiDiy2026

# 初回セットアップ
python _setup.py

# 全サーバー起動
python _start.py

起動後に確認するURL:

  • フロントエンド: http://localhost:8090/ ログイン: admin / ********
  • Core API: http://localhost:8091/docs
  • Apps API: http://localhost:8092/docs
起動できないとき
  • ポート競合 → 既に python.exe や node.exe が動いていないか確認
  • ログイン失敗 → パスワードは ******** (大文字小文字に注意)
  • DB異常 → backend_server\_data\AiDiy\database.db を削除して再起動

停止方法: python _stop.py または Ctrl+C で全サーバー停止
