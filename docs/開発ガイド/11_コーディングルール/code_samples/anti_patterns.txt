■ 1. 平文パスワードの本番使用
 絶対ダメ（本番環境）
パスワード = "********" # 平文保存

 本番ではハッシュ化必須
from passlib.context import CryptContext
pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")
パスワードハッシュ = pwd_context.hash(パスワード)

■ 2. 日本語コンポーネントタグ
 絶対ダメ（無効なHTML）
<C利用者一覧 />

 動的コンポーネント使用
<component :is="C利用者一覧" />

■ 3. GET/PUT/DELETEメソッドの使用
 誤り（このプロジェクトの規約違反）
@router.get("/一覧")
@router.put("/変更")
@router.delete("/削除")

 正解（POST統一）
@router.post("/一覧")
@router.post("/変更")
@router.post("/削除")

■ 4. Database VIEWの作成
 誤り（このプロジェクトでは不使用）
CREATE VIEW V利用者 AS SELECT ...

 正解（生SQLクエリ）
sql = "SELECT C利用者.*, C権限.権限名 FROM C利用者 LEFT JOIN C権限 ..."
result = db.execute(text(sql))
