■ M商品 動作確認手順

1. 一覧画面
 [ ] /Mマスタ/M商品/一覧 に遷移できる
 [ ] 初期表示で /apps/M商品分類/一覧 が呼ばれ、商品分類プルダウンが表示される
 [ ] 再検索で /apps/V商品/一覧 が呼ばれる
 [ ] /apps/V商品/一覧 の payload に 商品分類ID / 件数制限 / 無効も表示 が含まれる
 [ ] 商品IDまたは商品名リンク押下で編集画面へ遷移
 [ ] フィルタ/ソート/ページングが動作
 [ ] 件数制限ONで limit=1000、OFFで limit=null
 [ ] 無効も表示ONで有効列が表示される

2. 新規作成
 [ ] 「新規」押下で /Mマスタ/M商品/編集?モード=新規 へ遷移
 [ ] 必須（商品ID, 商品名, 単位, 商品分類ID）バリデーションが動作
 [ ] 商品分類プルダウンは有効な分類のみ選択肢になる
 [ ] 保存で /apps/M商品/登録 が呼ばれる
 [ ] 一覧へ戻り、成功メッセージが表示される

3. 編集
 [ ] 一覧の商品IDクリックで /Mマスタ/M商品/編集?モード=編集&商品ID=... へ遷移
 [ ] 初期表示で /apps/M商品/取得 が呼ばれ値が表示される
 [ ] 保存で /apps/M商品/変更 が呼ばれる

4. 削除
 [ ] 削除ボタンで qConfirm が表示される
 [ ] OK で /apps/M商品/削除 が呼ばれる
 [ ] 一覧へ戻り、削除メッセージが表示される
 [ ] 削除後は通常一覧から消え、無効も表示ONで有効=falseとして確認できる
