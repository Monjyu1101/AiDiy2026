一覧画面（M<テーブル名>一覧.vue）の構造概要

一覧画面は 親コンポーネント です。 ツールバー（再検索・新規）、チェックボックス（件数制限・無効も表示）、 一覧テーブル子コンポーネントを組み合わせて構成します。

要素説明
ツールバー左「再検索」ボタン → handleReload()
ツールバー右「新規」ボタン → openCreate() で編集画面へ
チェックボックス件数制限・無効も表示（変更時に handleReload()）
一覧テーブル<component :is="..."> で子コンポーネントをマウント
タイトルバー右端戻URLがある場合のみ「戻る」ボタンを表示

件数制限・無効も表示チェックボックスの UI パターン

件数制限 と 無効も表示 は ref() で管理し、 チェックボックス変更時に子の loadData() を呼び直します（handleReload）。 有効列表示 は computed で 無効も表示 と連動します。 

// <script setup> 内
const 件数制限 = ref(true);      // true: 上限1000件, false: 全件
const 無効も表示 = ref(false);   // false: 有効レコードのみ, true: 無効も含む
const 有効列表示 = computed(() => 無効も表示.value); // 無効も表示ONのとき「有効」列を表示

テンプレート部分（M車両一覧.vue より抜粋）:

<div class="table-options">
  <label class="checkbox-label">
    <input type="checkbox" v-model="件数制限" @change="handleReload" />
    件数制限
  </label>
  <label class="checkbox-label">
    <input type="checkbox" v-model="無効も表示" @change="handleReload" />
    無効も表示
  </label>
</div>

子コンポーネントへの props 渡しパターン

Vue のコンポーネントタグ名には ASCII 文字のみ使用できます。 日本語名のコンポーネントは <component :is="..."> 構文で使います。 

// import と ref
import 車両一覧テーブル from './components/M車両一覧テーブル.vue';
const 車両一覧テーブルRef = ref(null);
<!-- template 内 -->
<!--
  :件数制限     — true: 上限1000件取得, false: 全件取得
  :無効も表示   — APIリクエストに渡す（false: 有効のみ, true: 全レコード）
  :有効列表示   — 無効も表示ONのとき「有効」列をテーブルに表示する
  :戻URL        — 行クリック時に編集画面へ渡す戻URL（編集完了後にここへ戻る）

  ※ Vue コンポーネントタグ名は ASCII のみ有効。
    日本語コンポーネントは <component :is="..."> で使用する。
-->
<component
  :is="車両一覧テーブル"
  ref="車両一覧テーブルRef"
  :件数制限="件数制限"
  :無効も表示="無効も表示"
  :有効列表示="有効列表示"
  :戻URL="編集戻URL"
/>

handleReload — 子の loadData() を呼ぶパターン

「再検索」ボタンやチェックボックス変更時は、親から子の loadData() を呼び出します。 子コンポーネントは defineExpose({ loadData }) で公開し、 親は ref 経由でアクセスします。 

// 親（M車両一覧.vue）
const 車両一覧テーブルRef = ref(null);

const handleReload = () => {
  車両一覧テーブルRef.value?.loadData();
};
// 子（M車両一覧テーブル.vue）
defineExpose({
  loadData
});

一覧テーブルコンポーネント（M<テーブル名>一覧テーブル.vue）の構造

props 定義

const props = defineProps({
  件数制限: { type: Boolean, default: true },
  無効も表示: { type: Boolean, default: false },
  有効列表示: { type: Boolean, default: false },
  戻URL: { type: String, default: '' }
});

columns computed — 有効列の動的 splice

有効列表示 prop が true のときだけ「有効」列を splice で挿入します。 columns を computed にすることで、prop 変化に連動して自動的に列が表示/非表示になります。 

const columns = computed<Column[]>(() => {
  const baseColumns: Column[] = [
    { key: '車両ID',       label: '車両ID',       width: '120px', sortable: true, align: 'center' },
    { key: '車両名',       label: '車両名',       width: '200px', sortable: true },
    { key: '車両備考',     label: '車両備考',     width: '220px', sortable: true },
    { key: '更新日時',     label: '更新日時',     width: '160px', sortable: true },
    { key: '更新利用者名', label: '更新利用者名', width: '130px', sortable: true }
  ];
  if (props.有効列表示) {
    // 「車両備考」の前（index=3）に「有効」列を挿入
    baseColumns.splice(3, 0, { key: '有効', label: '有効', width: '60px', sortable: true, align: 'center' });
  }
  return baseColumns;
});

loadData — V系API 呼び出し

一覧取得は V系API (/apps/V<テーブル名>/一覧) を POST で呼びます。 リクエストボディに 件数制限 と 無効も表示 を渡します（M車両一覧テーブル.vue より抜粋）。 

const loadData = async () => {
  message.value = '';
  try {
    const res = await apiClient.post('/apps/V車両/一覧', {
      件数制限: props.件数制限,
      無効も表示: props.無効も表示
    });
    if (res.data.status === 'OK') {
      const data = res.data.data;
      const items = Array.isArray(data) ? data : data?.items ?? [];
      車両一覧.value = items;
      serverTotal.value = Array.isArray(data) ? items.length : Number(data?.total ?? items.length);
      currentPage.value = 1;
    } else {
      setMessage(res.data.message || 'M車両一覧の取得に失敗しました。', 'error');
    }
  } catch (e) {
    setMessage('M車両一覧の取得でエラーが発生しました。', 'error');
  }
};

有効列のセル表示 — qBooleanCheckbox

boolean 値の表示には qBooleanCheckbox コンポーネントを使います。

import qBooleanCheckbox from '../../../_share/qBooleanCheckbox.vue';

// template の #cell スロット内
<template v-else-if="column.key === '有効'">
  <qBooleanCheckbox :checked="Boolean(row.有効)" ariaLabel="有効状態" />
</template>

defineExpose

// 親から loadData() を呼び出せるように公開
defineExpose({
  loadData
});
