01_明日のために！その１_環境構築ハンズオン

このハンズオンの目的

この章は、workspace/AiDiy2026 をローカルで動かすための「最低限の仕組み」を、コードを読みながら理解するための資料です。

  • _setup.py が何を準備するか説明できる
  • _start.py が何を起動し、どう監視するか説明できる
  • 設定ファイル（backend_server/conf/、frontend_server/vite.config.ts）の役割を説明できる
  • トラブル時の確認ポイントを順序立てて実行できる

前提

  • 対象フォルダは workspace/AiDiy2026
  • Python 3.13 系と Node.js 22 系を利用
  • Windows / macOS / Linux いずれでも読める内容（コマンド例は Windows を中心に記載）

読み進め方

  1. 01_全体像で全体フローを把握する
  2. 02_ setup.py 解説でセットアップ処理を理解する
  3. 03_ start.py 解説で起動・監視・停止の仕組みを理解する
  4. 04_設定ファイルで設定値の関連を押さえる
  5. 05_まとめで確認と次の学習に進む
