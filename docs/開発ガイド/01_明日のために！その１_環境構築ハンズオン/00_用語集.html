00_用語集

このページは、環境構築ハンズオンで頻出する用語の最小辞典です。迷ったらここに戻って意味を確認してください。

開発環境とツール

用語意味補足
uvPython 依存関係を高速に扱うツールuv sync で pyproject.toml から環境を構築
npmNode.js のパッケージ管理ツールnpm install で package.json の依存関係を取得
venvPython の仮想環境本プロジェクトは通常 backend_server/.venv を利用
Viteフロントエンド開発サーバー兼ビルドツール開発時のポートは 8090
UvicornFastAPI を動かす ASGI サーバーcore:8091 / apps:8092

ネットワークとAPI

用語意味補足
ポート通信先を識別する番号このプロジェクトは 8090 / 8091 / 8092 を使用
localhost自分自身のPCを指すホスト名127.0.0.1 と同義
Proxyリクエストを別サーバーに転送する仕組みVite が /core と /apps を振り分ける
CORS異なるオリジン間通信を制御する仕組み許可設定が不足すると API がブラウザで失敗する
Swagger UIAPI をブラウザで試せる画面http://localhost:8091/docs と http://localhost:8092/docs

OSとプロセス

用語意味補足
プロセス実行中プログラムの単位Python / Node.js が別プロセスで動く
PIDプロセスIDOSが各プロセスに割り当てる番号
netstatポート使用状況を確認するコマンドWindows では netstat -ano が基本
taskkillWindows でプロセスを停止するコマンドtaskkill /F /T /PID <PID>
Ctrl+C実行中プロセスを停止する操作_start.py を終了するときに使う

データベース

用語意味補足
SQLiteファイルベースのRDBDBファイルは workspace/AiDiy2026/backend_server/_data/AiDiy/database.db
SQLAlchemyPython の ORMモデル定義と SQL 発行を扱う
AlembicDB マイグレーションツール本プロジェクトの標準運用では未使用
create_all()モデルからテーブルを作成する処理サーバー起動時の初期化で利用

認証

用語意味補足
JWTトークン形式の認証情報ログイン後に発行され API 呼び出し時に利用
SECRET_KEYJWT の署名に使う秘密鍵漏洩するとトークン偽造リスクがある
ALGORITHM署名アルゴリズム通常は HS256

よく使うコマンド

コマンド目的
python _setup.py初回セットアップを対話形式で実行
python _start.pyバックエンド2系統とフロントエンドを一括起動
cd backend_server && uv syncバックエンド依存関係を再同期
cd frontend_server && npm installフロントエンド依存関係を再インストール

確認ポイント

  • ポート番号は 8090 / 8091 / 8092 で統一されているか
  • パスは workspace/AiDiy2026 基準で読めているか
  • 「なぜこのツールを使うか」を説明できるか
