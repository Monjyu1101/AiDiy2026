99_まとめ

このハンズオンで確認した内容を、実務で使える形で最終整理します。

要点の総復習

  1. _setup.py は「環境の準備」担当
  2. _start.py は「起動・監視・停止」担当
  3. 設定ファイルは backend_server/conf/ と frontend_server/vite.config.ts を最初に確認
  4. 開発ポートは 8090 / 8091 / 8092 をセットで理解する
