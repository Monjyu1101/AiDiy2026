07_Router作成

Routerの実装ポイント

  • Depends(deps.get_db) でDBセッションを受け取る
  • Depends(deps.get_現在利用者) でJWT認証済み利用者を取得する
  • 重複/未検出は status="NG" と error.code で返す
  • 更新時は Optional項目のみ反映する
  • 削除エンドポイントは物理削除ではなく論理削除（有効 = False）で実装する
  • V系・T系の一覧は Optional[schemas.ListRequest] = None を受け取る。M系・C系は固定上限（MAX_ITEMS = 10000）でよい

論理削除パターン（M系・T系共通）

物理削除（db.delete(item)）は使いません。有効 フラグを False にする論理削除で実装します。

@router.post("/削除", response_model=schemas.ResponseBase)
def delete_M商品(
    request: schemas.M商品Delete,
    db: Session = Depends(deps.get_db),
    現在利用者: models.C利用者 = Depends(deps.get_現在利用者)
):
    """商品を論理削除（有効フラグをFalseに設定）"""
    item = crud.get_M商品(db, request.商品ID)
    if not item:
        return schemas.ResponseBase(
            status="NG",
            message="指定された商品が見つかりません",
            error={"code": "NOT_FOUND"}
        )

    認証情報 = {"利用者ID": 現在利用者.利用者ID, "利用者名": 現在利用者.利用者名}
    更新項目 = crud.create_audit_fields(認証情報, is_update=True)
    item.有効 = False  # 物理削除ではなく論理削除
    for key, value in 更新項目.items():
        setattr(item, key, value)
    db.commit()
    db.refresh(item)

    return schemas.ResponseBase(
        status="OK",
        message="商品の有効をオフにしました",
        data=schemas.M商品.from_orm(item)
    )

注意: 現在の code_samples/M商品_router.py は論理削除（有効=False）へ更新済みです。新規 Router でも同じパターンを使ってください。

一覧エンドポイントの件数制限（V系・T系）

V系・T系の一覧エンドポイントは list_controls.py の apply_active_filter を使ってORM queryにフィルタを適用できます。

from list_controls import apply_active_filter, get_list_limit

@router.post("/一覧", response_model=schemas.ResponseBase)
def list_T配車(
    request: Optional[schemas.ListRequest] = None,
    db: Session = Depends(deps.get_db),
    現在利用者: models.C利用者 = Depends(deps.get_現在利用者)
):
    query = db.query(models.T配車)
    query = apply_active_filter(query, models.T配車, request)  # 有効フィルタ
    total = query.count()
    limit_value = get_list_limit(request)
    items = query.order_by(models.T配車.配車伝票ID)
    if limit_value:
        items = items.limit(limit_value)
    items = items.all()

    return schemas.ResponseBase(
        status="OK",
        message="配車一覧を取得しました",
        data={"items": [schemas.T配車.from_orm(i) for i in items], "total": total, "limit": limit_value}
    )

apps_main.py 登録

from apps_router import M商品
app.include_router(M商品.router)

prefixはRouter側で持つため、include_router で追加prefixを重ねないでください。
