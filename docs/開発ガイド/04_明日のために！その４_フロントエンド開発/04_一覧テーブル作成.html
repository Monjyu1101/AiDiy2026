04_一覧テーブル作成

一覧テーブルは表示専用コンポーネントとして分離します。検索・ソート・ページングをここで扱います。

対象ファイル

workspace/AiDiy2026/frontend_web/src/components/Mマスタ/M商品/components/M商品一覧テーブル.vue

主な処理

  • POST /apps/V商品/一覧 で一覧データを取得
  • 列定義（商品ID、商品名、単位、商品備考、更新日時、更新利用者名）
  • フィルタ・ソート・ページング計算
  • 商品IDクリックで編集画面へ遷移
  • 件数制限・無効も表示 を props で受け取りAPIリクエストへ含める
  • 有効列表示 props に応じて「有効」列を動的に追加する

件数制限・無効も表示の props 定義と API 呼び出し（実装必須）

一覧テーブルコンポーネントは親から 件数制限、無効も表示、有効列表示 を受け取ります。 実際の実装は C利用者一覧テーブル.vue を参照してください。

props 定義

<script setup lang="ts">
const props = defineProps({
  件数制限: { type: Boolean, default: true },
  無効も表示: { type: Boolean, default: false },
  有効列表示: { type: Boolean, default: false },
  戻URL:    { type: String,  default: '' }
});
</script>

列定義への「有効」列の動的追加

const columns = computed<Column[]>(() => {
  const baseColumns: Column[] = [
    { key: '商品ID',       label: '商品ID',       width: '100px', sortable: true, align: 'center' },
    { key: '商品名',       label: '商品名',        width: '200px', sortable: true },
    { key: '単位',         label: '単位',          width: '80px',  sortable: true },
    { key: '商品備考',     label: '商品備考',      width: '250px', sortable: true },
    { key: '更新日時',     label: '更新日時',      width: '160px', sortable: true },
    { key: '更新利用者名', label: '更新利用者名',  width: '130px', sortable: true }
  ];

  // 無効も表示がONのとき「有効」列を挿入する（備考列の後ろなど任意位置でOK）
  if (props.有効列表示) {
    baseColumns.splice(4, 0, {
      key: '有効', label: '有効', width: '60px', sortable: true, align: 'center'
    });
  }

  return baseColumns;
});

API 呼び出し（件数制限・無効も表示をリクエストボディへ含める）

const loadData = async () => {
  try {
    const res = await apiClient.post('/apps/V商品/一覧', {
      件数制限: props.件数制限,
      無効も表示: props.無効も表示
    });
    if (res.data.status === 'OK') {
      const data = res.data.data;
      const items = Array.isArray(data) ? data : data?.items ?? [];
      rows.value = items;
      serverTotal.value = Array.isArray(data) ? items.length : Number(data?.total ?? items.length);
      currentPage.value = 1;
    }
  } catch (e) {
    // エラー処理
  }
};

defineExpose({ loadData });

「有効」列のセル描画

有効フラグは boolean 値なので、チェックボックス表示コンポーネント（qBooleanCheckbox）を使います。 

<template #cell="{ row, column, value }">
  <template v-if="column.key === '有効'">
    <qBooleanCheckbox :checked="Boolean(row.有効)" ariaLabel="有効状態" />
  </template>
  <template v-else>
    {{ value ?? '' }}
  </template>
</template>
