04_明日のために！その４_フロントエンド開発

この章の目的

この章では、workspace/AiDiy2026/frontend_server に M商品 の一覧・編集画面を追加し、バックエンドAPI（8092）と接続する流れを学びます。

  • Vue 3 のコンポーネント構成（一覧/編集）を理解する
  • Vue Router で画面遷移とクエリパラメータを扱う
  • API呼び出し（POST中心）で CRUD を実装する
  • 一覧テーブル・メニュー登録・動作確認まで完了する

前提

  • プロジェクトルート: workspace/AiDiy2026
  • 実装規約参照: workspace/AiDiy2026/frontend_server/AGENTS.md
  • フロントエンド: 8090、Apps API: 8092
  • バックエンド側で M商品 API が実装済み
  • 開発サーバー起動: python _start.py
  • VueコンポーネントタグはASCII名を使う（日本語名は <component :is="..." /> で扱う）
