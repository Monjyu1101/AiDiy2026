このシステムは何をする？

AiDiyは「管理画面中心の業務アプリ」をサクッと構築できる仕組みです。日本語のテーブル/JSON/URLで、現場の言葉にそのまま合わせて使います。

フロント

Vue 3 + Pinia。画面は管理向けに集約。ログイン後のメニューから各マスタへ移動。

http://localhost:8090 日本語UI

バックエンド

FastAPI + SQLite。APIは原則POST。日本語キーで統一されたレスポンス。

http://localhost:8091 JWT認証
