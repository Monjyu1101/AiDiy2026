■ 最短ルート（5ステップ）

1. 起動
   - `python _start.py` を実行

2. 稼働確認
   - ブラウザで `http://localhost:8090`
   - `http://localhost:8091/docs`（Core API）
   - `http://localhost:8092/docs`（Apps API）

3. ログイン
   - `admin / ********`
   - 失敗したら DB初期化条件（admin未存在時のみ投入）を確認

4. 画面遷移確認
   - Mマスタ → M商品 → 一覧 を開く

5. 最小CRUD確認
   - 一覧表示
   - 新規作成
   - 編集保存
   - 削除
