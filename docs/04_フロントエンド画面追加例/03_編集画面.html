この章のゴール

新規・編集・削除の3操作で `/apps/M商品/*` API が正しく呼ばれる状態にする。

編集画面の特徴

  • `モード=新規` なら新規モード、`商品ID` ありなら編集モード
  • 保存時は `/apps/M商品/登録` または `/apps/M商品/変更` を使い分け
  • 削除は `/apps/M商品/削除`（qConfirmで確認）
  • 保存/削除後は一覧へ戻してメッセージ表示
