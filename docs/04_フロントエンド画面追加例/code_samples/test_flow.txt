■ 動作確認手順

1. 一覧画面
   [ ] カテゴリメニューからカードをクリックして一覧画面に遷移
   [ ] データが正しくテーブルに表示される
   [ ] ソート機能が動作する
   [ ] ページング機能が動作する

2. 新規作成
   [ ] 「新規作成」ボタンをクリック
   [ ] 編集画面が「新規作成モード」で表示される
   [ ] 必須項目の入力チェックが動作
   [ ] 保存後、一覧に戻る
   [ ] 新しいデータが一覧に表示される

3. 編集
   [ ] 一覧で行をダブルクリック
   [ ] 編集画面が「編集モード」で表示される
   [ ] 既存データが入力欄に表示される
   [ ] 変更後、保存で一覧に戻る
   [ ] 変更内容が反映されている

4. 削除
   [ ] 編集画面で「削除」ボタンをクリック
   [ ] 確認ダイアログが表示される
   [ ] 「OK」で削除、一覧に戻る
   [ ] 削除されたデータが一覧から消えている

5. キャンセル
   [ ] 「キャンセル」ボタンで一覧に戻る
   [ ] 変更は保存されない
