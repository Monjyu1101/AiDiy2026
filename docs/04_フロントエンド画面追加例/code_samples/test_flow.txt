■ M商品 動作確認手順

1. 一覧画面
   [ ] /Mマスタ/M商品/一覧 に遷移できる
   [ ] 再検索で /apps/V商品/一覧 が呼ばれる
   [ ] 商品IDリンク押下で編集画面へ遷移
   [ ] フィルタ/ソート/ページングが動作

2. 新規作成
   [ ] 「新規」押下で /Mマスタ/M商品/編集?モード=新規 へ遷移
   [ ] 必須（商品ID, 商品名, 単位）バリデーションが動作
   [ ] 保存で /apps/M商品/登録 が呼ばれる
   [ ] 一覧へ戻り、成功メッセージが表示される

3. 編集
   [ ] 一覧の商品IDクリックで /Mマスタ/M商品/編集?モード=編集&商品ID=... へ遷移
   [ ] 初期表示で /apps/M商品/取得 が呼ばれ値が表示される
   [ ] 保存で /apps/M商品/変更 が呼ばれる

4. 削除
   [ ] 削除ボタンで qConfirm が表示される
   [ ] OK で /apps/M商品/削除 が呼ばれる
   [ ] 一覧へ戻り、削除メッセージが表示される
