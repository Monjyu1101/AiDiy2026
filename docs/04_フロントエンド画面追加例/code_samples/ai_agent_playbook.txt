■ AIエージェント作業ルール（最小版）

1. 変更単位
- 1ステップ = 1責務（例: 一覧テーブルだけ）
- まとめて広範囲を変更しない

2. 検証単位
- 変更した責務ごとに動作確認
- APIパスと画面遷移を必ず確認

3. 報告フォーマット
- 変更したファイル
- 変更理由
- 確認した動作
- 未確認/既知リスク

4. 迷った時の優先順位
- 実装コード（frontend_server/src）
- AGENTS.md
- docs/04 の順

5. 禁止事項
- /core と /apps を混在させる推測実装
- 日本語URL/日本語キーの勝手な英語化
- qAlert/qConfirm を native alert/confirm に置換
