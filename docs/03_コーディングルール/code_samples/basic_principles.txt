■ AiDiyの基本原則

1. 日本語ファースト
   - 画面/URL/JSON/DBの識別子は日本語が基本
   - システム用語や一般名詞は英語

2. UTF-8固定
   - 全ファイルはUTF-8エンコーディング必須
   - BOMなし推奨

3. POST統一
   - API操作は原則POSTメソッド
   - GET/PUT/DELETEは使わない

4. 統一レスポンス
   - status: "OK" / "NG"
   - message: メッセージ
   - data: 結果データ

5. 監査フィールド必須
   - 登録日時/登録利用者ID/登録利用者名/登録端末ID
   - 更新日時/更新利用者ID/更新利用者名/更新端末ID
