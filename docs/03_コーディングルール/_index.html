日本語ファースト設計とは

AiDiyでは、ビジネスドメインに関連する識別子は日本語を使用します。これは「課題」や「問題」ではなく、意図的な設計です。

日本語を使う

  • テーブル名: C権限, M商品, T配車
  • カラム名: 利用者ID, 配車日付
  • API endpoint: /core/利用者/一覧
  • JSON keys: 利用者名, 権限ID
  • Vue component名: C利用者一覧.vue

英語を使う

  • フレームワーク用語: router, store
  • 一般名詞: request, response, query
  • システム変数: items, total, limit
  • ライブラリAPI: ref, computed
