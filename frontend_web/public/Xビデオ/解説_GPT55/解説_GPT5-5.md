# 解説_GPT5-5
テーマ: OpenAI が2026年4月に公開した GPT-5.5 は、GPT-5.4 と同等のレイテンシを維持しながらフロンティア性能を大幅に向上させた最新モデルである。主な特徴として、エージェント型コーディング（Terminal-Bench 2.0: 82.7%、SWE-Bench Pro: 58.6%、Expert-SWE: 73.1%）、ナレッジワーク（GDPval: 84.9%、OSWorld-Verified: 78.7%、Tau2-bench Telecom: 98.0%）、科学研究（GeneBench: 25.0%、BixBench: 80.5%）、長文コンテキスト（1M トークン対応）、サイバーセキュリティ（CyberGym: 81.8%）、抽象的推論（ARC-AGI-2: 85.0%）での高い性能が挙げられる。ChatGPT および Codex（Plus/Pro/Business/Enterprise）で提供開始。API 料金は入力 $5/1M トークン、出力 $30/1M トークン。本解説では、GPT-5.5 の主な機能と、AiDiy の AIコア / Code AI に gpt-4.1 / gpt-5.5 を組み込んだ活用シナリオをわかりやすく解説する。
