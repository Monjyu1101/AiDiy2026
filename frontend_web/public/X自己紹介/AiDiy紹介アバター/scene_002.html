日本語ファースト設計

DB / API / UI / Code を 日本語で一直線につなぐ

AiDiy では業務語彙と実装名をできるだけ一致させ、日本語話者がコードと画面を直接対応づけられるようにしています。

UTF-8 固定
URL も日本語
JSON キーも日本語
一般名 request/query は英字可

一致重視

Database

  • テーブル名 `C権限`
  • カラム名 `利用者ID`
  • 監査フィールド標準

API / JSON

  • `/core/利用者/一覧`
  • `{"利用者名": "admin"}`
  • POST 中心の統一レスポンス

Frontend

  • `C利用者一覧.vue`
  • `/C管理/C利用者/一覧`
  • Vue Router / Pinia 運用

Code

  • `利用者名`
  • `配車日付`
  • `商品名`

確認ポイント

画面、URL、JSON キー、識別子は日本語を原則とする。

DB 項目名、API 項目名、フロントエンド変数名はできるだけ一致させる。

HowTo や検証手順は AGENTS.md ではなく .aidiy/knowledge に置く。
