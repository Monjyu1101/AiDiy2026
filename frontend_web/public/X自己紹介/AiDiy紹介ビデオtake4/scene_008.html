knowledge

  • `.aidiy/knowledge/_index.md` が全 HowTo の入口
  • 実装手順・検証手順を集約
  • AGENTS.md には書かない

AGENTS.md

  • 全体アーキテクチャ・設計方針
  • 各サブプロジェクトに配置
  • Code AI が最初に参照

人向け入口

  • `docs/` — 開発ガイド HTML
  • `X自己紹介/` — 紹介ビデオ・アバター
  • Web UI: http://localhost:8090
