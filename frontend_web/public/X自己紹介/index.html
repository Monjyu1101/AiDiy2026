Project Core

AiDiy の輪郭は、業務実装と AI 実装を同じ土台に置くこと。

権限管理やマスタ管理のような実務寄りの基盤から、音声対話、VRM アバター、コード支援まで、ひとつの流れで試せるように全体を組んでいます。

Architecture

構成は分かれていても、体験はひとつに寄せる。

backend_server

FastAPI + SQLAlchemy + SQLite。POST 中心 API、統一レスポンス、C採番、監査フィールド、Reboot 機構を標準装備。

core_main.pyapps_main.pyNo Alembic

frontend_web

Vue 3 + Vite + TypeScript。日本語ルーティング、qTubler、共通ダイアログ、CRUD パターンを維持します。

Vue RouterPiniaqTubler

frontend_avatar

Electron と通常ブラウザの両対応。BroadcastChannel で状態同期し、AI コアをマルチパネルで扱います。

ElectronWeb ModeBroadcastChannel

Advanced Layer

AI コア、音声対話、3D アバターまで踏み込む。

音声リアルタイム対話とアバター表現を、業務 UI と同じプロジェクトで試せます。

音声対話インターフェース

16kHz 入力 PCM を WebSocket で AI コアへ送り、24kHz 出力をキュー再生。口パク連動まで一貫して扱います。

Gemini LiveOpenAI RealtimeAudioController
