# AiDiy 登場企業・プロダクト・風袋画像URL一覧

X自己紹介ページの「Powered by」バッジや関連技術の装飾画像として並べるための候補。
各企業・プロダクトの公式ブランドアセットから引用（favicon / ロゴ SVG / ブランドガイドライン / 製品ロゴ）。

---

**採用戦略（優先順位）:**
1. **simpleicons.org:** 25件 — `https://cdn.simpleicons.org/<slug>`（ブラウザからは正常動作）
2. **公式favicon/ロゴ:** 4件 — Claude Code / VRM / Vue Router / Monaco Editor
3. 公式ページ内から探索
4. 代用不可　掲載見送り

## 補足

- ✅ = アイコン取得可能、❌ = 掲載見送り（採用戦略の優先順位で代用不可と判断）
- アイコンサイズは 24×24px または 32×32px で統一想定
- 実際の配置時は主要なものから優先的に採用し、数が多すぎる場合はカテゴリ代表のみ表示
